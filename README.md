# Turn-Based Combat Game

My first Python project using Object-Oriented Programming.

## How to Play
Run the script and choose a character to fight the boss Zioles.
- Attack, heal, or use your ultimate ability and pray to get critical hits
- Beat the boss before he beats you!

## Features
- 5 unique characters with different stats
- Boss AI that adapts based on health
- Ultimate ability system with charge meter
- Critical hit mechanics
